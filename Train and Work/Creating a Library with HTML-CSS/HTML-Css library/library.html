5 Margin & Padding 

                        
                             HTML
                             < div class="margin-padding">
                                < div class="margins">
                                    < p class="m5 bg-lightgreen">Margin 5

                            < div class="paddings">
                                    < p class="p5 bg-darkgreen">Padding 5
                            CSS
                            //.m5 {
                                margin:60px;
                            }
                        
                            .p5 {
                                padding:60px;
                            }//

Margin dış kenarlardan uzaklık değeri verirken padding ise border ile iç kısma ait değeri temsil eder. mb margin bottom, mt margin top gibi kullanımların kısaltması olarak bu kütüphanede kullanılmıştır. Ayrıca m5 ile doğrudan her kenardan eşit uzaklığı belirtmek mümkündür. Yukarıda hem margin hem padding için doğrudan m5 ve p5 değerleri kullanılmış ve çıktılara ait görünümler önce açık yeşil ardından koyu yeşil renkte gösterimler ile yansıtılmıştır.

6.Slider Çeşitleri

                        
                             HTML
                             < div class="slider1">
                                < div class="slider1-content">
                                    < div class="slider1-item">
                                        < img src="./assets/images/slider1.jpg" alt="">
                                    
                                    < div class="slider1-item">
                                        < img src="./assets/images/slider2.jpg" alt="">
                                    
                                    < div class="slider1-item">
                                        < img src="./assets/images/slider3.jpg" alt="">
                                    
                                    < div class="slider1-item">
                                        < img src="./assets/images/slider4.jpg" alt="">
                                    
                                
                            
                
                            CSS
                            //.slider1{
                                position: relative;
                                width:500px;
                                height:300px;
                                border: 2px solid #ccc;
                                margin: auto;
                                overflow: hidden;
                            }
                            .slider1-content{
                                position:absolute;
                                width: 400%;
                                left:0px;
                                animation: slide1 10s infinite;
                            }
                            .slider1-item{
                                width: 25%;
                                float: left;
                            }
                            .slider1-content:hover{
                                animation-play-state: paused;
                            }
                            
                            @keyframes slide1 {
                                0% {
                                    transform: translateX(0px);
                                }   
                                15% {
                                    transform: translateX(-500px);
                                }
                                40% {
                                    transform: translateX(-1000px);
                                }
                                50% {
                                    transform: translateX(-1000px);
                                }
                                65% {
                                    transform: translateX(-1500px);
                                }
                                75% {
                                    transform: translateX(-1500px);
                                }
                                90% {
                                    transform: translateX(0);
                                }
                            } //

Sliderlar websitelerinde farklı amaçlar için kullanılır ve dikkat çeken bölümleri oluştururlar. Temaya bağlı olarak animasyonlar ile sağdan sola, soldan sağa ya da yukarıdan aşağıya çeşitli türlerde tercih edilebilir. Yukarıda sağdan sola slider örneğinin kodu mevcuttur bu koda ait çıktı ise aşağıda ilk sırada görülmektedir. Slider üstüne gelindiğinde pause özelliği devreye girmektedir. Yazı bulunan bir sliderı okumak için kullanışlı bir özellik olacaktır.

9 Buton Yapıları

                         < button class="button-basic">Gönder
                         < button class="button-red">Gönder
                         
                        CSS
                        //
                        .button-basic{
                            width: 80px;
                            height: 40px;
                            padding: 10px;
                            padding-bottom: 10px;
                            color: rgb(0, 0, 0);
                            background-color: rgb(250, 246, 246);
                            border: 1px solid rgb(0,0,0);
                            border-radius: 10%;
                            box-shadow: 0 0 20px rgba(black);
                        }
                        .button-basic:hover{

                        .button-red{
                            width: 80px;

Butonlar kaydet, gönder gibi komutlara dikkat çekmek amaçlı kullanılmaktadır. Burada yukarıda kodu verilen butonlar normalde yansıtmakta oldukları rengin 0.2 değerinde transparanlıkta bulunurken hover efekti ile üstüne gelindiğinde 1 değerinde transparanlığa yani opaklığa çevrilirler. 10 farklı buton rengi ve 2 farklı buton tipi aşağıda yan yana olacak şekilde görülmektedir.

10 Resimler

                    
                         HTML
                         < div class="centered-image">
                            < img src="./assets/images/library.jpg" alt="Resim">
                        

                        CSS
                        //
                        .centered-image {
                            display: flex;
                            justify-content: center;
                            align-items: center;
                            width: 400px;
                            height: 300px; 
                            margin: 10px;
                        }
                          
                          .centered-image img {
                            max-width: 100%;
                            max-height: 100%;
                          }

                        //

Resimler için overflow bir div oluşturuldu. Ardından bu div içerisine eklenecek resimleri ortalı ve yüksekliği tam oturacak bir şekilde yerleştirecek bir class oluşturuldu. Resmin ortalı bir şekilde görüneni, resmin yüksekliğe göre ortalananı, resmin genişliğe göre ortalananı ve resmin orjinal boyutunda içeride overflow:hidden olarak görünen toplam 4 class mevcuttur. Sırasıyla aşağıda çıktıları verilen kodun centered haline ait kod yukarıda görülmektedir.

13 Form Elemanları

                    
                         HTML
                         < form>
                            < input type="text" class="custom-form-element" placeholder="Adınız">
                            < input type="email" class="custom-form-element" placeholder="E-posta Adresiniz">
                            < select class="custom-form-element">
                                < option value="option1">İstanbul
                                < option value="option2">Ankara
                                < option value="option3">İzmir
                            
                            < textarea class="custom-form-element" rows="4" placeholder="Mesajınız">
                            < button type="submit" class="custom-form-element">Gönder
                        
                        CSS
                        //
                        .custom-form-element {
                            display: block;
                            width: 100%;
                            padding: 10px;
                            margin-bottom: 10px;
                            border: 1px solid #ccc;
                            border-radius: 5px;
                            font-family: Arial, sans-serif;
                            font-size: 16px;
                          }
                        
                          .custom-form-element:hover{
                            background-color: gray;
                          }
                        //

Form elemanları kullanıcının giriş yapması, seçim yapması gibi çeşitli ögeleri içinde bulunduran yapılardır. Metin alanları sayesinde kullanıcılar yazı yazabilir, butonlar ile yazmış ya da seçmiş oldukları itemleri kaydedip, gönderebilirler. Bu bölümde form alanına ait kod yukarıda çıktısı ise aşağıda görülmektedir.
